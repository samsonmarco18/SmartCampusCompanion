Smart Campus Companion

App Description
Smart Campus Companion is an Android mobile application developed as part of an Android Development case study. The app is designed to assist university students by providing access to campus information through a simple and user-friendly mobile interface.

The project is developed using "Kotlin" and "Jetpack Compose" and follows a team-based development approach using "Git and GitHub". The application evolves across different academic periods, simulating a real-world software development process.

Git Workflow
Branches
- `main` – Stable and final version of the application
- `develop` – Active development branch

Workflow Process
- The project repository is initialized on GitHub
- All development is done on the `develop` branch
- Features are committed regularly with meaningful messages
- Pull Requests are used to merge changes
- Only the Team Leader merges changes into `main`
- Each team member contributes "at least 5 commits"
